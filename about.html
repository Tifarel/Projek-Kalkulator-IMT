 Web Programmer:

Seorang web programmer adalah individu yang bertanggung jawab untuk merancang, membuat, dan memelihara situs web dan aplikasi web. Mereka menggunakan berbagai bahasa pemrograman seperti HTML, CSS, JavaScript, dan bahasa server-side seperti PHP, Ruby, Python, atau Java untuk mengembangkan fungsi dan fitur situs web.

Content Maker:

Content maker adalah seseorang yang menciptakan konten untuk berbagai platform, baik itu teks, gambar, video, atau audio. Mereka dapat bekerja di berbagai media seperti blog, media sosial, YouTube, podcast, dan lainnya. Tujuan mereka adalah menghasilkan konten yang menarik dan relevan untuk audiens tertentu untuk meningkatkan engagement dan pengikut.

Graphic Designer:

Graphic designer adalah profesional yang menggabungkan seni dan teknologi untuk menciptakan berbagai bentuk komunikasi visual. Mereka bekerja dengan elemen visual seperti gambar, teks, dan grafik untuk membuat desain untuk media cetak dan digital, termasuk logo, brosur, majalah, iklan, dan antarmuka situs web.

Game Programmer:

Game programmer adalah orang yang mengembangkan kode untuk permainan video. Mereka menggunakan berbagai bahasa pemrograman dan alat pengembangan game untuk membuat mekanika permainan, kecerdasan buatan, grafik, dan suara yang diperlukan untuk membuat permainan yang fungsional dan menarik. Mereka bekerja sama dengan desainer game, artis, dan tim lain untuk menghidupkan konsep permainan.
